description: how to start the whatsapp clone project
---

To start the project anytime you come back, follow these steps:

1. **Open your terminal** (e.g., PowerShell or Command Prompt).
2. **Navigate to the server directory**:
   ```powershell
   cd 'c:\Users\devst\OneDrive\Documents\node-projects\whatsapp_chat_antig\server'
   ```
3. **Start the server**:
   // turbo
   ```powershell
   npm start
   ```
4. **Access the application**:
   Open your browser and go to:
   [http://localhost:5001](http://localhost:5001)

> [!TIP]
> Make sure your MongoDB service is running in the background for the project to connect to the database successfully.
